👋 嗨！这是一枚为你（也为我自己）写的极简 Windows 定时睡眠/关机小工具。
它够安静、够干净，而且只有一个文件。

## 🤔 为什么会有这个小玩意？

你有没有过这样的尴尬：深夜想听点播客、白噪音入睡，或者电脑正挂着下载大文件。你想设定一个时间让电脑自动睡觉，但在网上一搜，下载下来的软件不是要么觉得用不习惯，要么觉得功能太多太乱。

我只是想安安静静地设个倒计时，然后闭眼睡觉。
既然找不到称心的，那就自己写一个吧。于是，“极简睡眠”诞生了。

## ✨ 它能为你做些什么？（核心特性）

- **🟢 只有一个干干净净的 .exe**
  没有烦人的安装向导，不需要你自己去配置什么 .NET 运行环境（我都帮你打包压进这一个文件里了）。下载下来，扔在桌面上，随点随开。
  
- **🛡️ 睡梦中绝不“掉链子”**
  很多简单的倒计时软件有个致命 Bug：如果中途你的电脑自己息屏休眠了，倒计时就会卡死，醒来时间全乱了。我给它加了点“黑科技”（绝对时间锚点），无论电脑中间睡了多久，唤醒时依然精准无比。
  
- **🖐️ 护眼，更“防手残”**
  深夜友好的青蓝冷色调。输入框做了严格的限制，你没办法瞎输字母，全输 0 也会自动帮你纠正。内置了 30、60、120 分钟快捷键，懒人福音，点一下“开始”，世界就清静了。
  
- **🔔 给自己留点“后悔药”**
  倒计时结束时，它不会粗暴地直接断电。它会悄悄弹出一个永远在最前面的小窗口，给你 30 秒的考虑时间。如果 30 秒内你没有碰它？那晚安，它帮你关机。
  
- **🔒 绝不“精神分裂”**
  鼠标不听使唤双击了好几次？别担心，它底层加了一把互斥锁。它不会傻乎乎地打开一堆窗口自己跟自己打架，只会把已经打开的那个乖乖叫回屏幕最前面。

## 🚀 拿走就用 (Quick Start)

### 给普通用户：
1. 去右侧的 **[Releases](#)** 页面。
2. 下载 `极简睡眠.exe`。
3. 双击运行，设置好时间，安心睡觉吧！

### 给想折腾代码的同好：
如果你想在这个基础上改点东西，它的代码结构非常简单清晰。

**环境依赖：**
- Windows 操作系统
- .NET 8.0 SDK
